Vous avez à disposition une sandbox (bac à sable) dans laquelle vous pouvez construire un automate et simuler son exécution pour un mot donné en entrée (input). Pouvez-vous construire un automate qui accepte le langage régulier donné ci-dessous ?

[01]*1[01][01]

In other words, a word is part of that language if the third-to-last character is a 1, and all the rest is either a 0 or a 1.

State ◯

  • Add: double-click on the canvas
  • Move: drag it around
  • Delete: click it and press the delete key (not the backspace key)
  • Make it accepting: double-click it
  • Name: click it and type something
    • Numeric subscript: put an underscore before the number (like "S_0")
    • Greek letter: put a backslash before it (like "\beta")

Transition →

  • Add:
    • Start transition: shift-drag from the canvas to a state
    • Link states: shift-drag from one state to the other
    • Self-loop: shift-click on a state
  • Move: drag it around
  • Delete: click it and press the delete key (not the backspace key)
